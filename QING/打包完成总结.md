## 📦 打包方式

### 方式一: 使用 Android Studio (推荐)

#### 快速测试版 (Debug APK)
```
1. 打开 Android Studio
2. 点击 Build → Build Bundle(s) / APK(s) → Build APK(s)
3. 等待构建完成
4. APK 位置: app/build/outputs/apk/debug/app-debug.apk
```

#### 正式发布版 (Release APK)
```
1. 打开 Android Studio
2. 点击 Build → Generate Signed Bundle / APK
3. 选择 APK → Next
4. 创建或选择签名密钥
5. 选择 release 构建类型
6. 点击 Finish
7. APK 位置: app/build/outputs/apk/release/app-release.apk
```

### 方式二: 使用命令行

#### Debug 版本
```bash
cd DrawConnect
gradlew assembleDebug
```

#### Release 版本
```bash
cd DrawConnect
gradlew assembleRelease
```

### 方式三: 使用批处理脚本 (Windows)

```bash
# 双击运行
构建APK.bat

# 然后选择:
# [1] 构建 Debug APK
# [2] 构建 Release APK
# [3] 清理构建缓存
```
